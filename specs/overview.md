# Hackathon Todo - Project Overview

## Project Description
A full-stack todo application with AI-powered chatbot capabilities that allows users to manage their tasks through both a traditional web interface and natural language interactions.

## Target Phases
- **Phase II**: Full-Stack Web Application with authentication and CRUD operations
- **Phase III**: AI-Powered Todo Chatbot with MCP tools integration

## Core Functionality
- User authentication and authorization
- Task management (Create, Read, Update, Delete, Complete)
- AI chatbot for natural language task management
- Responsive web interface with modern UX

## Technology Stack
- Backend: FastAPI, SQLModel, Neon PostgreSQL
- Frontend: Next.js 16+, TypeScript, Tailwind CSS
- Authentication: Better Auth
- AI: OpenAI Agents SDK, MCP SDK
